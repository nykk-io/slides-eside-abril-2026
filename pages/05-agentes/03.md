# Anatomía mínima de un agente

```
LOOP {
  1. observa()     // ¿qué hay, qué pasó?
  2. piensa()      // ¿qué paso me acerca al objetivo?
  3. actua()       // llama una tool: bash, browser, editor, API
  4. evalua()      // ¿funcionó?
  5. ¿terminé?     // salir. Si no → volver al 1.
}
```
